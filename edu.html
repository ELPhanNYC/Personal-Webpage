Information on Relevant Courses

  • Advanced Placement Calculus BC
    • Course covers college-level calculus 1 and 2.
    • Topics include Differentiation and Integration as well as there applications, Differential Equations, Polar Coordinates, and Parametric Equations.
  • CSE115: Introduction to Computer Science I
    • Course provides students with an introduction to programming and computer science through Python and JavaScript using Repl.it as the primary code editor.
    • Topics taught in class include variables, conditionals, loops, function, control flow, sorting, reading/writing files, databases (SQLite3 via Python), basic algorithm design, web security, and web development (JSON and Bottle Web Framework).
    • Students are tasked with creating a web application at the end of the semester using the topics discussed in class.
  • CSE116: Introduction to Computer Science II
    • Course dove deeper in the field of computer science as students were taught different paradigms used in software development.
    • Students are taught programming paradigms such as object-oriented programming and functional programming, first-order/higher-order functions, memory usage and management (stack, heap, pointers), runtime complexity and a introduction to advanced data structures and algorithms.
    • Students are tasked with completing a project related to each of the topics above.
  • CSE191: Discrete Mathematics
    • Course provides students with foundation logic to be used in the field of computer science.
    • Topics taught in lecture include logic, proofs, sets, functions, relations, recursion, recurrence relations, mathematical induction, graphs, trees, basic counting theory, regular expressions, and finite automata.
  • CSE199: University at Buffalo Computer Science Seminar
    • Focuses on introducing students into the areas of study in Computer Science.
    • Allows students to connect with each other and meet various faculty members through the style of the course.
    • Topics include privacy, misinformation, racial and gender bias in algorithms, and data representation.
  • STA301: Probability and Statistics
    • Introduces students to Set Theory, Counting Methods, Conditional Probability, Random Variables, and Discrete,Continuous, and Multivariate Random Variables
  • MTH309: Introduction to Linear Algebra
    • Course cover the principles of linear algebra such as row elimination (Row and Reduce-Row Echelon Form), Determinants, Inverse Matrices, System of Linear Equations, Matrix Transformations, Linear Transformations, Vector Spaces, Least Square Solutions, Gram-Schmidt Process, and Eigenvectors and Values.
